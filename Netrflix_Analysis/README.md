# Netflix Data Analysis

## Project Overview
Explored the Netflix dataset to analyze the distribution of content (Movies vs. TV Shows) and identify top content-producing countries.

## Tech Stack
- Python (Pandas, Matplotlib)
- Data Manipulation: Filtering and counting unique values.

## Key Insights
- Content Ratio: Movies constitute a significantly larger portion of the library compared to TV Shows.
- Top Countries: United States and India are the leading producers of content on the platform.
- Trend: Analyzed the release years to see how content volume has grown over time.

## What I Learned
- Using value_counts() for frequency distribution.
- Filtering data to isolate specific types of content.
- Creating Pie Charts to show market share.
