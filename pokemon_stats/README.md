# Pokemon Stats Analysis

## Project Overview
Analyzed a dataset of Pokemon to find the strongest, fastest, and most legendary characters. The project focused on sorting and multi-condition filtering.

## Tech Stack
- Python (Pandas)
- Logic: Boolean Indexing and Conditional Logic.

## Key Insights
- Strongest Type: Dragon-type Pokemon have the highest average Attack stats.
- Fastest Pokemon: Identified characters like Deoxys (Speed Forme) and Ninjask as top speedsters.
- Legendary Analysis: Filtered and counted Legendary vs. Non-Legendary Pokemon.

## What I Learned
- Using AND (&) and OR (|) operators for complex queries.
- Sorting data using sort_values().
- Comparing groups using groupby.
