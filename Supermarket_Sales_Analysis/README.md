# Supermarket Sales Analysis

## Project Overview
Analyzed a retail dataset to understand sales trends across different branches and product lines. The goal was to identify high-performing categories and payment methods using Python.

## Tech Stack
- Python (Pandas, Matplotlib)
- Data Cleaning: Handling dates and currency formats.

## Key Insights
- Identified the branch with the highest total revenue.
- Analyzed which 'Product Line' generates the most income.
- Visualized payment method preferences (Cash vs. Credit Card vs. E-wallet).

## What I Learned
- Loading CSV files using Pandas.
- Basic aggregation using groupby().
- Creating Bar Charts to visualize categorical data.
